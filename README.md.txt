## 📌 Project Overview  
This project is a **Mobile Sales Performance Dashboard** built in Power BI.  
It provides insights into key business metrics such as:  
- 📱 **Total Revenue & Sales Trends**  
- 🌍 **Regional Sales Distribution**  
- 🛒 **Top Products & Categories**  
- 📈 **Profitability Insights**  

The dashboard was designed to practice **data cleaning, data modeling, DAX measures, and visualization best practices** in Power BI.  

---

## 🎯 Key Features  
✅ Dynamic visuals with interactive filters  
✅ KPI cards for quick performance overview  
✅ Trend analysis across regions & categories  
✅ User-friendly layout for business stakeholders

## 🚀 How to Use  
1. Clone this repository or download the `.pbix` file.  
2. Open the file in **Power BI Desktop**.  
3. Interact with the dashboard using slicers and filters.
